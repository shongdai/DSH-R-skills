# 写作规则
- 导语拆短行：痛点（?）→ 方案（试试）→ 价值（！），每行一个信息单元
- 图片解读：每条独立一行，不用缩进续行，复杂描述拆为多条
- 全篇表情不重复，语义与文字协调匹配
- 只用常见 Unicode 表情，不用冷门或平台不兼容的
- 不用 ✓、▸ 等单一符号，用表情增强视觉层次
- 标题 ≤20 字，正文 ≤600 字
